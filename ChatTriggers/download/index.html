  • ChangeLog

    Array improvements and additions
      - {array[$arrayname]}.clear()
         - clears the entire array
      - {array[$arrayname]}.has($value)
         - returns true if $value is in $arrayname
         - otherwise returns false
      - {array[$arrayname]}.size()
         - returns number of entries in $arrayname


    Json support
      - {array[$arrayname]}.importJsonFile($dir,$node)
        - loads all nodes with value $node from the json in $dir into $arrayname
        - eg: {array[names]}.importJsonFile(./mods/ChatTriggers/names.json,name)
           if names.json looks like '{"name":"kerbybit"},{"name":"Ecolsson"},{"name":"Cham"}' it will load all of the values in "name" into the array "names"

      - {array[$arrayname]}.importJsonURL($url,$node)
        - Same thing as importJsonFile but from a url
        - eg: {array[uuid]}.importJsonURL(https://api.mojang.com/users/profiles/minecraft/kerbybit,id)
          This will get the uuid for kerbybit and store it in array uuid. You can use this to grab the usernames of kerbybit also

    support for layered nodes comming soon (eg: node(0)->name)
    Im also working on a php json creator for hypixel stats that will let you get stats with importJsonURL


    New trigger types
      - onWorldLoad
        - runs every time the world gets loaded (joining singleplayer, switching servers, joining a minigame)
      - onWorldFirstLoad
        - only runs the first time a world is loaded once mc starts up
        - great for loading a json file with a lot of data that you only need to load once
      - onServerChange
        - runs when you change servers
        - wont get ran when joining a minigame


    Misc
      - added {server} string -> returns current server or SinglePlayer
      - added <vol=$volume> and <pitch=$pitch> tags for sound event
      - removed legacy support for old tags


    Bug fixes
      - fixed logic functions breaking sometimes due to a copy paste mistake by me
      - fixed trigger arguments not working properly with logic functions
      - fixed trigger arguments not running using event:trigger $triggerName($arguments)
      - fixed commas in strings breaking trigger arguments
      - fixed .toUpper() and .toLower() breaking string interrupts
      - revamped clickable to be more reliable
      - many many many many more minor bug fixes
