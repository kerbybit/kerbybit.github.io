  • ChangeLog

    Complete string functions overhaul
      - String functions now get run left to right like expected
      - Anything that string functions can be run on (arrays, trigger arguments, default strings) now get temporary converted into strings so everything works flawlessly
      - String functions are infinitely stackable and nestable
      - Strings, themselves, are now infinitely nestable
      - String function errors now get handled properly
         - {string[$sn]} gets replaced in event with the error
         - this allows for some clever error handling by the users

    New string functions
      - {string[$sn]}.add($num) - adds $num to $sn given that they are both numbers
      - {string[$sn]}.subtract($num) - subtracts $num from $sn
      - {string[$sn]}.multiply($num) - multiplies $sn by $num
      - {string[$sn]}.divide($num) - divides $sn by $num (returns int, not float)
      - {string[$sn]}.power($num) - raises $sn to the power $num

      - {string[$sn]}.set($value) - sets $value to $sn without saving to files
      - {string[$sn]}.save($value) - saves $value to $sn with saving to files
      - {string[$sn]}.importJsonFile($file,$node) imports first $node from $file
      - {string[$sn]}.importJsonURL($url,$node) imports first $node from $url

      - NOTE: string.set($sn,$value) and string.save($sn,$value) are deprecated!

    New built in strings
      - {server} - returns server name ('Single Player' if offline)
      - {serverIP} - returns server IP ('localhost' if offline)
      - {serverMOTD} - returns server MOTD ('Single Player world' if offline)
      - {serverversion} - returns server version ('1.8' if offline)
      - {ping} - returns ping to current server ('5' if offline)
      - {debug} - returns the state of debug mode (/trigger settings debug)

    Other various changes
      - notify now defaults to <pos={notifysize}> instead of 0
      - fixed bug with notify showing in upper left hand corner for a frame
      - added /trigger settings killfeed position [top-left/top-right]
      - added killfeed position to settings.txt
      - improved json importing
