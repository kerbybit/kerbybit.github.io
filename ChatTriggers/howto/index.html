Overview

ChatTriggers is a dynamic and powerful mod that YOU control!
Using simple commands, you can change chat just the way you want it! Play a sound when you throw someone off the edge of the map, get rid of pesky "sale" messages in chat, and even completely rearange how your online friends list looks! anything is possible!

Chat has never been this much fun!
Run commands, play sounds, create warning messages, cancel chat events. With loads of trigger events, anything is possible! Want a twitch like bot for your house? No problem! Want a "remove friend when they log off" button? Piece of cake!

Triggers

Creating a trigger
Command
/trigger create [trigger type] [trigger]
Command usage
Create a trigger that trips any events tied to it
Trigger types
chat - gets tripped when a chat message matches the trigger

other - only gets tripped on other trigger events





.

Trigger tags
About
Trigger tags are used inside of triggers to organize them and show special attributes about them

Tags are always surrounded by '{' and '}'

Tags
{list='listName'}
The list tag is used to organize triggers. This will help you keep things tidie as you add more and more triggers
{s} {c} {e}
These are used to tell ChatTriggers where in the message it is looking for text. {s} is for if the message starts with the specified trigger, {c} is for if it contains the trigger, and {e} is for if it ends with the trigger












NOTE: Want to make sure you are picking up the right color in your trigger? just add the formatting codes!
Remember, if you have one formatting code (eg: &5), if you dont have all of the formatting codes, the trigger WONT get tripped.
you can use '/trigger setting debug' to see all of the color formatting codes the same way the mod sees them

Events

Creating an event
Command
/trigger add [trigger number] [event type] [event arguments]
Command usage
Create an event for when a trigger gets tripped



.

Event types
Chat
Shows a client side message of the event arguments in chat

clickable(text,click event,click value,hover text)
Creates clickable chat!

text: the text that you can click on

click event: the type of click event (eg: suggest_command, run_command, open_url)

click value: the value for the click event (eg:/party invite NoxyD)

hover text (optional): text that is displayed on hover
Say
Makes you say the event arguments in chat
Command
Runs the command that is in the event arguments
Killfeed
Shows event arguments in the killfeed
Cancel
Cancels the chat message that triggered
Trigger
Trips another trigger, triggering its events
Choose
Randomly chooses one of any events below it







Strings

Creating a string
Command
/trigger string create [string name]
Command usage
Create a string

.

Reading a string
{string<string name>}
This gets replaced with what is stored inside of the string

This can be used in either an trigger or an event
Built in strings
{me}
Your username
{msg}
The chat message that tripped the trigger
{trigsize}
Returns how many triggers you have

.

Setting a string with a command
Command
/trigger string set [string number] [string]
Command use
Sets the string to what you want

.

Setting a string with a trigger
{string[string name]}
If this is inside of a trigger, the trigger will attempt to set the string if every thing else in the trigger matches.

This is a very powerful feature and extremely fun to use!
{string[string name]number}
Checks to make sure that if a string is about to be set, it needs to be a certain number of words long.

If the number of words doesnt match the number specified, the string doesnt get set.
Examples
/trigger create {string[name]1} is in {string[game]}

/trigger create Guild > {string[name]}: {string[msg]}

/trigger create {s}{string[name]1} killed
